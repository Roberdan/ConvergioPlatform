# ADR-004: MyConvergio Transformation to Community Marketplace

**Status**: Accepted | **Date**: 2026-03-22

**Context**: Core platform in ConvergioPlatform. MyConvergio had duplicated/diverging content.
**Decision**: MyConvergio becomes community skills marketplace. License changed to Apache 2.0. copilot-agents/ removed (replaced by transpilers).
**Consequences**: Clear separation of concerns. Community contributions via Skill Protocol.
